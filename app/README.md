# Prother - Voice Prompts for AI Coding

Speak your coding prompts instead of typing them. Works with GitHub Copilot, Cline, Claude Code, and more.

## Features

- **Voice Input**: Press `Alt+V` or say "Hey Prother" to start speaking
- **AI Enhancement**: Make your prompts better with AI (Gemini free tier)
- **Universal**: Works with any AI coding assistant
- **Free**: Uses free Gemini API (~1,500 enhancements/day, no credit card)
- **Private**: No backend, no tracking, keys stored in OS keychain

## Quick Start

1. Install this extension
2. Click "Start Setup" in the welcome notification
3. Get a free Gemini API key from [Google AI Studio](https://aistudio.google.com/app/apikey)
4. Paste your key and start speaking

## Development

```bash
cd app
npm install
npm run build
# Press F5 in VS Code to launch Extension Development Host
```
